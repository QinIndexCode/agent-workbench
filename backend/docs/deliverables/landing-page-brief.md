# 产品落地页文案大纲

## Assumptions

- **产品类型未知**：假设这是一个面向企业或个人用户的 SaaS / 数字产品，具有明确的核心价值主张。
- **目标市场**：假设主要面向中文用户市场（中国大陆及华语地区）。
- **品牌调性**：假设品牌风格为专业、可信赖、现代感，避免过度花哨或过于保守。
- **页面目标**：假设落地页的核心目标是引导用户注册 / 试用 / 购买，而非单纯品牌曝光。
- **竞品环境**：假设市场中存在同类竞品，需要突出差异化优势。
- **技术实现**：假设页面为单页滚动式（Single Page Application），适配桌面端和移动端。

## Audience

### 主要受众

| 受众类型 | 特征 | 核心需求 |
|---------|------|----------|
| 企业决策者 | 关注 ROI、效率提升、团队协作 | 解决业务痛点，降低运营成本 |
| 产品 / 技术负责人 | 关注集成能力、稳定性、可扩展性 | 技术可靠，易于集成 |
| 终端用户 | 关注易用性、功能完整性 | 上手简单，功能满足日常需求 |

### 用户痛点

1. 现有方案效率低下，耗时耗力
2. 工具分散，数据不互通
3. 学习成本高，团队推广困难
4. 价格不透明，隐性成本多
