# 定时任务

定时任务页用于在 SCC **保持打开** 时按计划创建任务。它适合重复性工作，但不是操作系统后台常驻服务。

## 这页会影响什么

- 任务会在什么时间或间隔被触发
- 任务默认使用哪个文件夹上下文
- 计划失败时你是否能在界面中看到结果或错误

## 两种计划方式

### 按固定时间

适合每天、每周或每月在固定时刻执行。

### 按间隔时间

适合在应用打开期间按固定频率重复运行。

## 内置 reflection 任务

系统默认会保留一个 reflection 任务，用于整理近期任务记忆、模式和潜在 skill 候选。

它：

- 可以暂停
- 可以编辑
- 不提供普通删除按钮

## 推荐的首次验证

建议先创建一个低风险测试计划，确认：

1. 时间预览是否符合预期
2. 计划是否只在 SCC 打开时运行
3. 暂停和恢复是否立即生效
4. 结果是否真的进入任务流
5. 失败信息是否能在列表中看到
