# 偏好

偏好页负责个人使用习惯和本地行为，不直接替代权限审批。

## 这页不会做什么

偏好页不会：

- 自动放宽工具权限
- 绕过审批模式
- 让外部工具或搜索直接获得更高权限

如果你想控制风险边界，应去 **权限审批**。

## 个性化设置

这里主要调整：

- 语言
- 主题
- Agent 语气
- 回答详略
- 启动页面
- Agent 角色
- Skill 注入数量
- 是否展示思考

## 行为设置

### 清理敏感数据

优先对常见密钥和敏感片段做脱敏处理，减少工具证据直接暴露敏感值的概率。

### 加密本地存储

启用后，本地 SQLite 记录的 payload 会按当前实现做加密存储。
