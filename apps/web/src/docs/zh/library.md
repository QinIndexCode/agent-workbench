# 资料库

资料库用于保存可复用背景材料，不是第二条任务时间线。

## 建议从哪里开始

1. 想看可复用执行方法时，先看 **Skills**
2. 想知道为什么被推荐、拦截或判重时，先看 **Curator**
3. 想管理说明文档、引用资料或检索质量时，先看 **知识库**
4. 想维护长期画像和项目记忆时，先看 **记忆**
5. 想审查晋升阶段和证据链时，查看 Curator 的运行记录

## 这里适合放什么

- 可复用的方法和约束
- 持久化项目事实
- 值得以后再检索的导入资料
- 解释晋升决策的 Curator 运行记录

不要把一次性任务输出直接堆进资料库，否则匹配和检索都会变差。

## 推荐操作顺序

1. 先把 **记忆** 保持准确和紧凑
2. 再整理 **知识库** 内容和标签
3. 然后到 **Curator** 看证据，再决定是否激活候选
4. 最后才在 **Skills** 内做激活、暂停或合并
