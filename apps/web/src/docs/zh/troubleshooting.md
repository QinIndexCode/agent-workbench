# 故障排除

## 模型无响应

- 检查 API Key 是否正确配置
- 确认网络连接正常
- 查看模型服务商的状态页面
- 检查是否达到速率限制

## 语音输入不可用

- 确认浏览器支持 Web Speech API
- 检查麦克风权限是否已授予
- 尝试刷新页面后重新授权

## 工具执行失败

- 检查当前权限级别是否允许该操作
- 确认工具参数是否正确
- 查看任务时间线中的错误信息

## 界面显示异常

- 尝试刷新页面
- 清除浏览器缓存
- 检查浏览器控制台是否有报错

## 数据存储

- 任务状态、模型/搜索/集成配置、附件、检查点、记忆和 Skill 由本机 Agent Workbench server 写入 SQLite。
- 浏览器本地存储主要保存界面偏好等 UI 状态。清除浏览器数据可能重置这些界面偏好，但不会删除 server SQLite 中的任务历史。
- API Key 和集成密钥会提交给本机 server，以加密 secret 引用保存；Web UI 只回显脱敏后的元信息，例如末四位。
- 默认 server 只面向可信本机访问，不要直接暴露到不可信网络。
