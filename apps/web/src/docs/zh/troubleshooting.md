# Troubleshooting

## 模型无响应

- 检查 API Key 是否正确配置
- 确认网络连接正常
- 查看模型服务商的状态页面
- 检查是否达到速率限制

## 语音输入不可用

- 确认浏览器支持 Web Speech API
- 检查麦克风权限是否已授予
- 尝试刷新页面后重新授权

## 工具执行失败

- 检查当前权限级别是否允许该操作
- 确认工具参数是否正确
- 查看任务时间线中的错误信息

## 界面显示异常

- 尝试刷新页面
- 清除浏览器缓存
- 检查浏览器控制台是否有报错

## 数据存储

- 所有数据存储在浏览器本地
- 清除浏览器数据会导致任务历史丢失
- API Key 存储在本地，不会上传到服务器
