# Permissions

权限系统用于控制工具调用的执行策略，按风险等级分级管理。

## 权限级别

### Ask
每次工具执行前请求用户确认。适用于：
- 处理敏感数据
- 执行可能影响系统的操作
- 需要了解每步操作细节的场景

### Read only
只读操作自动通过，写入操作仍需确认。适用于：
- 文件读取、目录浏览
- 信息查询类操作

### All
所有风险类别全局允许。适用于：
- 重复性、可预期的任务
- 已建立信任的工作流

可随时从 All 降级到更严格的级别。

## 风险分类

系统对工具调用进行以下分类：
- **低风险**：读取、查询、计算
- **中风险**：文件修改、配置变更
- **高风险**：删除、网络请求、系统命令
