# 输入方式

Agent Workbench 支持多种输入方式，用于向任务会话提供信息。

## 文本输入

- 输入框支持多行文本，按 `Shift + Enter` 换行
- 按 `Enter` 发送消息
- 支持 Markdown 格式的文本渲染

## 文件附件

- 点击输入框左侧的回形针图标上传文件
- 支持同时上传多个文件
- 文件内容会被读取并作为上下文的一部分发送给模型
- 支持的文件类型取决于浏览器和系统环境

## 语音输入

- 点击麦克风图标启用语音识别
- 使用浏览器的 Web Speech API，无需额外安装
- 识别结果会填入输入框，可编辑后发送
- 兼容性取决于浏览器是否支持 SpeechRecognition 接口
