# Task Management

任务是 SCC 的核心组织单元，每个任务对应一次独立的会话上下文。

## 创建任务

- 点击侧边栏的 **添加任务** 按钮创建新任务
- 新任务会出现在侧边栏的任务列表中
- 点击任务标题可切换当前会话

## 任务历史

- 所有任务会话会自动保存
- 侧边栏的任务列表支持滚动浏览
- 点击任务可查看完整的历史消息

## 任务执行流程

1. 用户发送消息
2. 系统组装上下文（历史消息 + 当前输入 + 附件内容）
3. 模型生成响应
4. 如响应包含工具调用，系统按权限策略处理
5. 工具执行结果返回模型，继续生成
6. 最终结果展示给用户
