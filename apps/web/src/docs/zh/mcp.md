# MCP

Model Context Protocol (MCP) 是一种开放协议，用于连接外部工具服务。

## 功能

- 统一的工具发现和调用接口
- 工具运行在独立进程中
- 动态工具列表同步

## 配置本地服务

1. 进入设置页面的 **MCP** 标签
2. 点击 **添加服务**
3. 填写服务名称、启动命令和参数
4. 保存后系统会尝试连接并获取工具列表

## 配置远程服务

- 支持 SSE (Server-Sent Events) 传输
- 填写服务端点后自动同步工具列表

## 使用

- 模型在需要时生成 MCP 工具调用
- 执行前按当前权限策略处理
- 结果追加到任务时间线
