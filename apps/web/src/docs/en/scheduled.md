# Scheduled Tasks

The Scheduled Tasks page creates tasks on a plan **while Agent Workbench is open**. It is useful for repeated work, but it is not an operating-system background service.

## What this page affects

- when recurring tasks are triggered
- which folder context they use
- whether failures are visible quickly enough to act on

## Two schedule modes

### Fixed time

Use this for daily, weekly, or monthly work.

### Interval

Use this for repeated work while the app remains open.

## The built-in reflection task

Agent Workbench keeps one default reflection task to maintain recent task memory, reusable patterns, and candidate skills.

That default task:

- can be paused
- can be edited
- intentionally does not expose the normal delete action
