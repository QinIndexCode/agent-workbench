# Preferences

The Preferences page is for personal workflow and local behavior. It is not a shortcut around approvals.

## What this page does not do

Preferences do **not**:

- loosen tool permissions automatically
- override approval modes
- grant more power to search, MCP, or integrations

## Personalization settings

This page is where you tune:

- language
- theme
- agent tone
- response detail
- startup view
- agent role
- max auto-injected skills
- whether thinking stays visible

## Behavior settings

### Sanitize sensitive data

Redacts common secrets and sensitive fragments where possible before they remain visible in tool evidence.

### Encrypt local storage

Encrypts locally stored SQLite payload records according to the current storage implementation.
