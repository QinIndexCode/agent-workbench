# Settings

The Settings page is used to manage workbench configuration.

## Providers

Manage model providers:
- Add, edit, and delete provider configurations
- Set the default model
- Specify context window size

## Permissions

Manage tool execution permissions:
- Set the global permission level
- Configure per-tool-type permissions

## MCP

Manage external tool services:
- Add/edit/delete MCP configurations
- View service status and tool lists

## Preferences

Adjust UI and notifications:
- Switch interface language
- Configure notification reminders
- Manage local data storage
