# Reflections

Reflections are the audit trail for promotion decisions.

## What a reflection tells you

- which stage Agent Workbench is in
- what evidence is already strong
- what still needs more successful examples
- whether duplicates or conflicts were detected

## Common stages

- collecting task memory
- extracting reusable skill candidates
- checking duplicate or conflict signals
- waiting for more successful examples

## Good use of this page

Use Reflections to understand why no skill appeared yet, or why a candidate remains blocked.
