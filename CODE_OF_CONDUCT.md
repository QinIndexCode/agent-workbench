# Code of Conduct

## Our commitment

We want SCC-Batch to be a collaborative, respectful, and technically honest project. Contributors should be able to ask questions, disagree about implementation details, and review code rigorously without being belittled.

## Expected behavior

- Be respectful and professional.
- Critique code and ideas, not people.
- Prefer curiosity and evidence over assumption.
- Help others understand context instead of gatekeeping it.

## Unacceptable behavior

- Harassment, intimidation, or discrimination
- Personal attacks or sustained hostility
- Deliberate disruption of collaboration spaces
- Publishing private or sensitive information without permission

## Enforcement

Project maintainers may remove or reject contributions, comments, or participation that violate this code of conduct.

## Reporting

Please report conduct concerns through the security/contact guidance in [`SECURITY.md`](./SECURITY.md) until a dedicated project contact is published.
